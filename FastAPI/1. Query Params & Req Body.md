`app/schemas.py`

```python
from pydantic import BaseModel 

class PostCreate(BaseModel):
    title: str 
    content: str 
    
```

- A “request body” is the JSON payload sent by the client in a POST request (not in the URL). In FastAPI, parameters that are Pydantic models like `PostCreate` are parsed from the body, not from query/path.

`app/app.py`

```python
from fastapi import FastAPI, HTTPException
from app.schemas import PostCreate 

app = FastAPI()

text_posts = {
    1: {"title": "First Post", "content": "This is the first post."},
    2: {"title": "Second Post", "content": "This is the second post."},
    3: {"title": "Learning Python", "content": "Python makes problem-solving simple and fun."},
    4: {"title": "Web Development", "content": "Building websites combines logic and creativity."},
    5: {"title": "Daily Coding", "content": "Small progress each day leads to big results."},
    6: {"title": "Debugging Tips", "content": "Errors are clues, not failures."},
    7: {"title": "Clean Code", "content": "Readable code saves time in the future."},
    8: {"title": "Version Control", "content": "Git helps track and manage changes efficiently."},
    9: {"title": "APIs Explained", "content": "APIs allow applications to communicate seamlessly."},
    10: {"title": "Frontend Basics", "content": "HTML, CSS, and JavaScript build user experiences."},
    11: {"title": "Backend Logic", "content": "Servers handle data and business rules."},
    12: {"title": "Stay Consistent", "content": "Consistency beats motivation every time."}
}

# e.g. /posts?limit=3 
@app.get("/posts")
def get_all_posts(limit: int):
    if limit:
        return list(text_posts.values())[:limit]
    return text_posts

@app.get("/posts/{post_id}")
def get_post(post_id: int):
    if post_id not in text_posts:
        raise HTTPException(status_code=404, detail="Post not found")
    return text_posts.get(post_id, {"error": "Post not found"})


# post: PostCreate tells FastAPI to expect a JSON body matching the PostCreate schema (e.g. {"title": "...", "content": "..."}).
# It builds new_post from the validated fields (post.title, post.content).
# It assigns a new ID by taking the max existing key + 1.
# It stores the new post and returns it as the response.
@app.post("/posts")
def create_post(post: PostCreate):
    new_post = {"title": post.title, "content": post.content}
    text_posts[max(text_posts.keys()) + 1] = new_post
    return new_post
```

# Common format specifiers to reference

| Specifier | Meaning              | Example |
| --------- | -------------------- | ------- |
| `%Y`      | 4-digit year         | `2025`  |
| `%y`      | 2-digit year         | `25`    |
| `%m`      | month (00–12)        | `07`    |
| `%c`      | month (1–12)         | `7`     |
| `%d`      | day of month (00–31) | `09`    |
| `%e`      | day of month (1–31)  | `9`     |
| `%H`      | hour (00–23)         | `14`    |
| `%h`      | hour (01–12)         | `02`    |
| `%i`      | minutes (00–59)      | `05`    |
| `%s`      | seconds (00–59)      | `07`    |
| `%p`      | AM or PM             | `PM`    |

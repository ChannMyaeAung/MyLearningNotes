# PostgreSQL Between

```bash
dvdrental=# SELECT payment_id, amount
dvdrental-# FROM payment
dvdrental-# WHERE payment_id BETWEEN 17503 AND 17505
dvdrental-# ORDER BY payment_id;
 payment_id | amount
------------+--------
      17503 |   7.99
      17504 |   1.99
      17505 |   7.99
(3 rows)

dvdrental=# SELECT payment_id, amount
dvdrental-# FROM payment
dvdrental-# WHERE payment_id NOT BETWEEN 17503 AND 17505
dvdrental-# ORDER BY payment_id;
 payment_id | amount
------------+--------
      17506 |   2.99
      17507 |   7.99
      17508 |   5.99
      17509 |   5.99
      ...
      ...
      ...
```

## With a date range

```postgresql
SELECT
	CUSTOMER_ID,
	PAYMENT_ID,
	AMOUNT,
	PAYMENT_DATE
FROM
	PAYMENT
WHERE
	PAYMENT_DATE BETWEEN '2007-02-15' AND '2007-02-20'
	AND AMOUNT > 10
ORDER BY
	PAYMENT_DATE;
```

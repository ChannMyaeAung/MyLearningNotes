# PostgreSQL IN

- Allows us to check whether a value matches any value in a list of values.

```bash
dvdrental=# SELECT film_id, title
dvdrental-# FROM film
dvdrental-# WHERE film_id in (1,2,3);
 film_id |      title
---------+------------------
       1 | Academy Dinosaur
       2 | Ace Goldfinger
       3 | Adaptation Holes
(3 rows)

dvdrental=# SELECT first_name, last_name
dvdrental-# FROM actor
dvdrental-# WHERE last_name IN ('Allen', 'Chase', 'Davis')
dvdrental-# ORDER BY last_name;
 first_name | last_name
------------+-----------
 Meryl      | Allen
 Cuba       | Allen
 Kim        | Allen
 Jon        | Chase
 Ed         | Chase
 Susan      | Davis
 Jennifer   | Davis
 Susan      | Davis
(8 rows)
```

# PostgreSQL NOT IN operator

```postgresql
SELECT
	FILM_ID,
	TITLE
FROM
	FILM
WHERE
	FILM_ID NOT IN (1, 2, 3)
ORDER BY
	FILM_ID;
```

# Arrays

**PostgreSQL**

- PostgreSQL supports **true array types** with querying and indexing capabilities.

```postgresql
CREATE TABLE teams (
  id SERIAL PRIMARY KEY,
  members TEXT[]
);

-- Insert
INSERT INTO teams (members) VALUES (ARRAY['Alice', 'Bob', 'Chan']);

-- Query
SELECT * FROM teams WHERE 'Chan' = ANY(members);
```

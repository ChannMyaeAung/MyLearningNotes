To start completely fresh (removing old containers, images, and volumes), we can do:

```bash
docker compose down -v  # Stops containers and removes volumes
docker system prune -a  # Removes unused images, containers, etc. (be careful, this cleans globally)
docker compose up --build
```

- But if we are just testing the changes, `docker compose down` && `docker compose up --build` is sufficient, it will rebuild the images and recreat
